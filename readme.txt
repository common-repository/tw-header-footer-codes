TW Header & Footer Codes is plugin that will help you add HTML, JavaScript of CSS codes to the head and footer part of your WordPress website.

== Description ==

= Add Header and Footer Codes/Scripts Easily =

Add custom HTML, JavaScript or CSS codes or scripts to your WordPress website with ease. You can easily add Google Analytics, Facebook Pixel tracking codes and other custom codes using this plugin.

= Features =

* Global header code injection, will load the codes on all pages inside <head> tag
* Global footer code injection, will load the codes on all pages before </body> tag
* Page/post specific header codes, will only load the codes on that that specific page only inside <head> tag
* Page/post specific footer codes, will only load the codes on that that specific page only before </body> tag
* Post type support, you can choose which post type will be able to add it's own header / footer codes (support custom post type)

== Installation ==

1. Put the plugin folder into WordPress /wp-content/plugins/ directory or install it from WordPress Dashboard by going to Plugins > Add Menu
2. Activate the plugin from WordPress dashboard Plugins menu.
3. Go to the Settings > TW Header& Footer Codes menu to configure the plugin

== Frequently Asked Questions ==

= Where are the plugin settings located? =

It's located under Settings > TW Header & Footer Codes menu.

= Can I use it to add tracking code? =

Sure, you can them using this plugin to add Google Analytics, HiStats, Facebook Pixel, Google AdWords Conversion or any other tracking code.

= How can I add code to specific page? =

First you need to enable the post type support from the plugin settings. Once that's covered you will see a new meta box to add header & footer codes from the add/edit screen.
